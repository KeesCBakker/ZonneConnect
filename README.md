# Zonneplan One solar panel stats to PVOutput

Work in progress. Written in .NET. Uses Docker to run.

To start this:

1. Run `bash connect.sh`
2. Enter your Zonneplan email, PVOutput ApiKey and SystemId

To update, run:

1. Run `bash run.sh`
